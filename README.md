# Sentiment Analysis Project

This project uses Python and NLTK to perform **sentiment analysis** on movie reviews.  
It uses the **Naive Bayes classifier** to predict whether a review is positive or negative.

## Features
- Train a classifier on the Movie Reviews dataset
- Test accuracy on unseen reviews
- Predict sentiment of any custom text

## Project Structure
- `src/sentiment.py` → Main code
- `README.md` → Project explanation
- `.gitignore` → Ignore cache files
